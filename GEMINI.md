# 프로젝트 규칙 (Project Rules)

## 프로젝트 개요
- 서비스명: Once, Again (가족 아카이브)
- 주요 기능: 가족 스토리 저장, 갤러리, 방명록, 향수 브랜드 소개

## 코딩 스타일 및 기술 스택
- Framework: Next.js (App Router 활용)
- Styling: Tailwind CSS (Utility-first 방식 고수)
- Language: TypeScript
- UI 가이드: 스크린샷에 보이는 것처럼 미니멀하고 깨끗한 디자인 톤 유지

## 특별 지시 사항
- 모든 텍스트와 주석은 한국어를 기본으로 한다.
- 이미지 경로는 `/public` 폴더 내의 리소스를 우선 사용한다.
- 반응형 레이아웃(모바일/데스크탑)을 항상 고려하여 Tailwind 클래스를 작성한다.
